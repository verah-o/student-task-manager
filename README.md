# Student Task Manager

## Description
A web app to help students manage their daily tasks. Users can add tasks, mark them as completed, and delete them. Tasks are saved in the browser even after refreshing the page.

## Features
- Add tasks
- Mark tasks as completed (single click)
- Delete tasks (double click)
- Persistent storage with localStorage

## Technologies
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event handling
- Working with localStorage
- Basic UI design principles
